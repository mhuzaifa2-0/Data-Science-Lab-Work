# Intermediate Project - Climate Change Analysis

## Description
This project analyzes climate change trends using historical data such as temperature and CO2 emissions to understand patterns and changes over time.

## Problem Statement
Analyze climate data to identify trends and patterns related to climate change.

## Dataset
- Source: Kaggle / Climate datasets
- Description: The dataset includes information such as temperature changes, CO2 emissions, and other environmental factors over time.

## Steps Performed
1. Data Cleaning
   - Checked for missing values
   - Removed unnecessary columns

2. Exploratory Data Analysis (EDA)
   - Analyzed temperature trends over years
   - Studied CO2 emission patterns

3. Data Visualization
   - Line plots for temperature changes
   - Graphs showing emission trends

4. Model Building (Optional)
   - Basic prediction using linear regression

## Results
- Observed increasing trend in global temperatures
- CO2 emissions show a rising pattern over the years

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Conclusion
The analysis shows clear evidence of climate change trends, with increasing temperatures and emissions over time.
