# Fun Project - Rock Paper Scissors Game

## Description
This is a simple Python game where the user plays Rock, Paper, Scissors against the computer.

## Problem Statement
Create a game where the computer randomly selects Rock, Paper, or Scissors and the user tries to beat it.

## Approach
- The computer generates a random choice.
- The user inputs their choice.
- The program compares both choices and determines the winner.
- The game can be played multiple times.

## Python Libraries Used
- random

## Built-in Functions Used
- input()
- print()

## Features
- Interactive gameplay
- Random computer choices
- Multiple rounds
- Input validation

## Conclusion
This project demonstrates Python basics like conditional statements, loops, and user interaction.
